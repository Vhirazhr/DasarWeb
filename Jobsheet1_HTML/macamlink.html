Link relatif (relative link) dalam HTML adalah cara untuk merujuk ke file, atau sumber daya (seperti halaman web lain, gambar, stylesheet, atau skrip) yang lokasinya relatif terhadap lokasi file HTML yang sedang dibuka. Berbeda dengan link absolut yang menggunakan path lengkap atau URL penuh, link relatif hanya mencantumkan sebagian dari path yang dimulai dari lokasi file HTML.

Ini adalah contoh Link Absolut

klik di sini untuk pencarian yang anda inginkan.

Link absolut (absolute link) dalam HTML adalah jenis link, yang mencakup URL lengkap yang merujuk ke sumber daya tertentu, seperti halaman web, gambar, atau file lain, dari mana saja di internet. Link absolut berisi semua informasi yang diperlukan untuk menemukan sumber daya, termasuk protokol (seperti http:// atau https:/), nama domain, dan jalur lengkap ke sumber daya.

"Link with new window" adalah istilah yang digunakan ketika membuat hyperlink di HTML yang, saat diklik, akan membuka halaman web atau sumber daya di jendela atau tab baru browser, bukan di jendela atau tab yang sama. Ini sering digunakan untuk membuka tautan eksternal atau halaman yang tidak ingin menavigasi menjauh dari halaman saat ini.

Ini adalah contoh link ke email

Anda bisa menghubungi saya di email ini untuk pertanyaan lebih lanjut.

"Link to email" atau "email link" dalam HTML adalah hyperlink yang, ketika diklik, membuka aplikasi email default pengguna (seperti Microsoft Outlook, Apple Mail, atau Gmail) dengan alamat email penerima yang sudah diisi di kolom "To". Ini mempermudah pengguna untuk mengirim email tanpa harus mengetikkan alamat email secara manual.
